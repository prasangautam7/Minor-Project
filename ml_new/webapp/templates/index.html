How It Works

1

Upload Data

Upload a CSV file with heart rate (BPM) and SpO2 measurements per second or per minute.

2

AI Analysis

Our Random Forest model extracts 19 clinical features from 60-second windows and predicts apnea probability.

3

Get Results

View detailed dashboard with severity, visualizations, and insights.

Required CSV Format

Per-second data (recommended): 

second,heart_rate_bpm,spo2
0,68,81
1,69,81
2,69,80
3,70,79
4,72,78
5,74,80
...

Per-minute data (also supported): 

minute,heart_rate_bpm,spo2
0,67,97
1,71,96
2,74,94
3,80,91
...

Required: Both heart_rate_bpm AND spo2 columns are needed. The model uses 19 features from HR variability, SpO2 desaturations, and their cross-correlation.

Optional: Add a label column (0=Normal, 1=Apnea) to see accuracy metrics.
